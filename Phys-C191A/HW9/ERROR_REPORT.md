## 发现的主要错误

### 1. **第2题：Shor码逻辑态定义错误**
**原始代码：**
```
$$ \ket{\overline{I}} = \left(\frac{\ket{000}-\ket{11}}{\sqrt{2}}\right)^{\otimes 3} $$
```
**问题：** 
- 用 $\ket{I}$ 表示逻辑态1不规范
- $\ket{11}$ 只有2个量子比特，应为 $\ket{111}$

**修正后：**
```
$$ \ket{\overline{1}} = \left(\frac{\ket{000}-\ket{111}}{\sqrt{2}}\right)^{\otimes 3} $$
```

---

### 2. **第2.5题：Case 2和Case 3的综合症相同**
**原始问题：**
- Case 2 (Y_1错误)：综合症 $(-1, +1, +1, +1, +1, +1, -1, +1)$
- Case 3 (Z_1错误)：综合症 $(+1, +1, +1, +1, +1, +1, -1, +1)$

**分析：**
- Y_1 = iX_1Z_1，应同时与Z型和X型稳定子反交换
- Z_1只与X型稳定子反交换，与Z型稳定子可交换
- 因此综合症应不同

**修正：**
- Case 2 (Y_1): 在Z_1Z_2位置为-1（正确）
- Case 3 (Z_1): 在Z_1Z_2位置为+1（正确）
- 这些综合症是可区分的
